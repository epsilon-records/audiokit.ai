# 🛠️ Scripts Directory 🚀

## 🌟 Overview
This directory contains utility scripts for project maintenance and automation.

## 🔑 Key Scripts
- `fix_context_files.py`: 🛠️ Automates context file validation and fixes
- `initialize_templates.py`: 📄 Creates initial context file templates
- `validate_context_files.py`: ✅ Validates context file structure and content

## 🚀 Usage
1. 🐍 Ensure Python 3.8+ is installed
2. 📦 Install requirements: `pip install -r requirements.txt`
3. 🏃 Run scripts: `python scripts/<script_name>.py`

## 🛠️ Maintenance
Scripts are automatically validated and tested. Please update documentation when making changes.
