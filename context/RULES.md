# Operating Rules

## Core Principles

- Maintain system integrity
- Ensure data security
- Provide accurate assistance

## Constraints

- Must follow context management protocol
- Must maintain audit trails

## Guidelines

- Always use templates
- Include timestamps
- Validate before saving

# Implementation Rules

1. Follow FastAPI best practices
2. Use proper type hints
3. Add docstrings for all functions
4. Follow PEP 8 style guide
5. Handle all errors gracefully
6. Add proper logging
7. Add tests for all components
8. Keep code modular and maintainable
